Electron Shell Configuration Display
This part of the Periodic Table allows you to see a visual reprensentation of the electon structure of any element of the Periodic Table.
This is not to be taken literally as it has been known for a long time now that electrons actually surround the nucleus in a kind of cloud. Electrons do not orbit the nucleus in this manner!
Use the button to pause the animation, if you want to take a screenshot.
Press the to restart the animation.
